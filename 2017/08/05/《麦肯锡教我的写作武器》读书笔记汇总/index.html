《麦肯锡教我的写作武器》读书笔记汇总

发表于 2017-08-05 | | 阅读次数

这本书从7月5日开始读起，7月12日结束。

Day1 《麦肯锡教我的写作武器》 高杉尚孝著

前几天买了一个课程，送了这本书，刚好作为闲书来看看。
这本书主要讲的是如何通过逻辑思考写出一个好文案（文章）。全书共七章，前半部分第一章至第三章先讲述基础：信息是什么，如何表达信息，如何设定信息；后半部分第四章至第七章以问题解决型故事展开法为中心，通过事例来讲解。
第一章 信息
信息可大致分为两类：what、what should be，细分分为三类：记述、评价、规范（应该怎么样）。向人传达信息时，从记述到评价到规范是递进的。视情况可以忽略规范甚至忽略评价，前提是很懂对方的思考逻辑，确信对方能自己搭建逻辑链从A到B。
主题是信息的容器，限定信息的传达范围。主题和信息在以下几方面要配套：范围、时间轴、印象、种类。

Day2 《麦肯锡教我的写作武器》第二章

这一章主要是讲如何清楚表达信息，可以从这三个方面入手：主语、连接词、具体性。
在单句里，主语和谓语的关系要清晰明了，要有意识地注意到行为者是谁（元认知）。每个句子之间的逻辑衔接要清楚表达。在描述时多使用具体性的文字更好。

Day3 《麦肯锡教我的写作武器》第三章

这一章主要是讲述如何设定信息，最重要的技巧是金字塔结构和MECE分析架构。
金字塔结构可由下而上，也可由上而下。在由下而上时需要精炼信息，这里的精炼可以是具象或抽象。具象是指精炼出的信息仍然是具体的，是一个结论性的概括；抽象是指将下层的信息抽象出共同特质，将概念层次向上提升一层。精炼时要特别注意适度精炼，不能过度。
MECE分析架构简单地说就是将如何将一个主题细分为几个子主题，即需要做到“不重复、不遗漏”。最好多多积累一些分类的模型（MECE模型），在使用金字塔表达信息时可以直接套用。

Day4 《麦肯锡教我的写作武器》第四章

这一章讲的如何解决问题，由于作者工作于咨询行业，第四章是以他的角度所写的，具体步骤不太适用于我。
解决问题五步骤：
1、发现问题。要对问题的类型做分类：恢复原状、预防隐患或追求理想。
2、界定问题。站在对方角度考虑。不同问题类型的重心不一样。
3、列举方案。可采用头脑风暴法。
4、评价方案。根据某种标准评价，风险也要评估。
5、实施解决策略。

Day5 《麦肯锡教我的写作武器》第五章

这一章讲的是学会用SCQOR法叙述一个解决问题的故事。这一章内容比较干，可以当作一个模型记下来以后用，感觉电影的叙事手法和这个很像。
S：situation（设定状况）。介绍主角和故事发生的背景（或者说范围），持续至今的稳定状况。
C：complication（发现问题）。确定问题的类型（恢复原状or预防隐患or追求理想）。
Q：question（设定课题）。根据问题的类型设定具体的课题。
O：obstacle（克服障碍）。回答在Q环节设定的课题。
R：resolution（解决收尾）。简要给出结论。
SCQ是故事的导入部分，篇幅较少，O占故事的六至七成（人们想看的是主角如何克服重重困难斩杀恶龙的故事）。

附问题类型对应的具体课题：
1 恢复原状型：
掌握状况、应急处理、分析原因、根本措施、防止复发。
2 预防隐患型：
假设不良状态、诱因分析、预防策略、发生时的应对策略。
3 追求理想型：
资产盘点、选定理想、实施策略。

Day6 《麦肯锡教我的写作武器》第六章

这一章是讲如何将金字塔结构应用于商务文案的两种代表格式：报告和简报。内容比较细致，不好用文字来描述，有部分内容值得借鉴，可以用于PPT的逻辑构造上。

Day7 《麦肯锡教我的写作武器》第七章 7.1～7.3

这一章介绍各种能让说服力提升的技巧。
信息传达，由上而下最好。高价商品或服务定位成能规避同额损失的预防隐患型问题策略比较好。文案定义的问题类型要符合对方认知。风险要说明，但是要传达收益比风险大的信息，并且要根据对方容许的风险范围说明。

Day8 《麦肯锡教我的写作武器》第七章 7.4～7.6

给替代方案的时候，最好给3个，并且按照上中下的顺序给出（心锚效应），将最想让对方接受的选项放在中间（人们喜欢中庸）。
规范信息要根据对方的行动原理（内在逻辑）来说明，规范命题有实利命题（对对方有利）和伦理命题（使对方遵守道德）两种，两相结合效果更好。对于记述信息，使用因果或实证方法来链接根据和结论；对于评价信息，要根据对方的评价标准来增加根据。
