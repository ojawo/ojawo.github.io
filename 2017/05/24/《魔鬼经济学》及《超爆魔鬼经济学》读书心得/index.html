《魔鬼经济学》及《超爆魔鬼经济学》读书心得

发表于 2017-05-24 | | 阅读次数

前言

《魔鬼经济学》及《超爆魔鬼经济学》这两本书都是由经济学家史蒂芬·列维特和新闻工作者史蒂芬·都伯纳写的，先不说这个第二作者了（通讯作者介绍了干嘛），我主要介绍一下第一作者史蒂芬·列维特吧。史蒂芬·列维特本人是在哈佛大学毕业，后又在麻省理工大学取得经济学博士学位，现在在芝加哥大学当经济学教授，他还获得了约翰·贝茨·克拉克奖（40岁以下经济学家的最高荣誉）。上面这一句话就是想证明这位高才生的智商是很高滴，千万别信他自己说的“我数学不好，微积分就考2分”啊，学霸总要谦虚谦虚嘛！

这个史蒂芬·列维特和一般的经济学家还不太一样，他不研究听起来很经济学的问题，比如金融货币之类的。他研究的近乎是一些鸡毛蒜皮、狗咬耗子的日常生活小事，我们从他写的书里面摘几句章节名看看：“为什么毒品贩子仍然跟自己的妈妈住在一起”、“街头妓女与百货商店圣诞老人有何相似”、“三K党跟房产经纪人有何共同之处”。他最著名的研究就是关于美国犯罪率下降和堕胎合法化的研究。从这些我们可以看出来了，列维特是跑到别人的地盘（社会学、心理学等等）用经济学的方法抢肉来了，用现在时髦的话说，没想到列维特也是一个跨界高手啊！学界有个不好听的词形容这一派经济学家，叫”经济学帝国主义“，窃以为还是很形象的。

这两本书我在自己看完后提炼出了三个关键词：动机、关联、信息。下面按点论述。

动机

根据维基百科的定义，经济学是一门对产品和服务的生产、分配以及消费进行研究的社会科学，经济学的核心思想是资源配置与优化再生。这听起来比较学术，有点抽象。史蒂芬·列维特在书里就给出了经济学的另一个定义。从根本上来说，经济学就是研究人的动机：人怎样得到自己想要或者是需要的东西，尤其是当其他人也想要或者需要同样东西的时候。这个定义比较好懂，也比较符合直觉。

举个栗子，A同学有一天想要一个东西，这个东西是A没有的，而恰好B同学有，所以A同学为了得到这个东西就会和B同学进行交换。用什么东西交换呢？可能是以A物易B物，也可能是以A和B都认同的等价物（比如货币）。这整个交换的行为就是经济学的一个缩影。如果只是一个个体与另一个个体之间交换，是很好理解的，那如果是无数个个体呢？无数个个体之间（通常以公司、国家为集合）彼此进行交换，那可就很复杂了，经济学就是试图研究这样行为的一门科学。

什么是动机呢？动机可以是你知道考到100分，老妈就给你买一台xbox one，你就会拼命学习；也可以是你害怕在课堂上挖鼻孔被暗恋女生看到，从而被鄙视，你就会在暗恋女生面前保持一个良好的形象；也可以是你因为超速行驶而得到了一张罚单，损失了宝贵的money，以后你就不会再想超速了。按我的理解，动机是一种类似于激励的东西，驱使人们去做或者不去做某些事情的原动力。总的来说，动机是一种促使人们多做好事，少做坏事的方式。动机分为三类：经济动机、社会动机和道德动机。

经济学家相信，只要能设立合理的激励惩罚机制，这个世界上所有的问题都能得到解决。敲黑板划重点，合理的激励惩罚机制！来个快速问答吧，这个社会最大的激励惩罚机制是什么？5秒钟不许想，立刻回答！当然是法律啦！法律就是规定哪些坏事不能做的东西，你做了是要受到惩罚滴！可惜法律并没规定给做好事的人奖励（悄悄问了“法律界专业人士”）。如果设立的激励惩罚机制不合理（或者说不健全），人们可能就会尝试欺骗，欺骗也是一种经济行为，定义为以更小的代价换取更大的回报。至于如何设立合理的激励惩罚机制，则需要读者思考了。

关联

万事万物都是有关联的，关联的形式可能非常复杂，可能不仅仅是因果关系。让我们想象一张有许多个结点互相连接起来的网，每个结点都与许多另外的结点相连，这些另外的结点又与其他结点相连。如果把这些结点看作一个个的事物（概念）或者事情，那么影响这个事物的关联因素就非常多，我们很难去发现隐藏在背后真正的原因。万幸我们现在有一个名为“科学”的武器，能够让我们客观地去研究这些关联。在这本书里反反复复提到了控制变量法，这个方法想必大家都不陌生，简而言之就是固定住其他变量，只让一个变量变化，看结果是否发生改变。

举个栗子吧，让我们以史蒂芬·列维特最著名的研究说起。1989年，美国的犯罪率达到了历史上的顶峰，在过去的15年里，美国的暴力犯罪率上升了80%，整个国家都在讨论暴力犯罪问题。但是20世纪90年代初期犯罪率开始下降，下降的速度和幅度同样出乎意料，以至于当时的专家没反应过来，仍然继续坚信犯罪率上升。许多年后，他们才意识到犯罪率下降这个事实，并急急忙忙给出各种解释。这些解释有：新型巡管政策、更加发达的监狱系统、强劲的经济增长、警力的增强等等，这些解释看上去都很有说服力（你有没有意识到这里所提出各种解释的动机呢？）。在《魔鬼经济学》里有详细分析各种解释是否成立，我就不在这里展开了。

史蒂芬·列维特提出了一种解释：美国犯罪率下降是因为1978年宣布的堕胎合法化，堕胎降低了潜在的罪犯数量。这是一种因果关系，如何证明堕胎和犯罪率之间确实是因果关系而非相互关联呢？答案是利用控制变量法。在美国宣布罗伊威德法令（即堕胎合法化）的前两年，已经有5个州可以合法堕胎了。事实上，这5个州的犯罪率下降也领先于美国其他45个州。继续分析发现，那些在20世纪70年代堕胎率最高的州往往犯罪率下降的幅度也是最高的（已经将其他变量进行了控制），还发现在20世纪80年代后半期之前，一个州的犯罪率跟该州的堕胎率之间并没有任何关系。这里已经可以看出堕胎合法化和犯罪率下降是因果关系了，书中还进一步地做了分析，我就到此为止，感兴趣的同学可以看《魔鬼经济学》第四章。

信息

信息是什么呢？我们每天刷微博看到的时事新闻、QQ群里的一条全体艾特、股票价格的变动等等都是信息（必须得区分信息和观点），我们想必都会承认，信息在这个互联网时代已经越来越重要了，信息就是互联网世界的流通货币。信息本身是中立的，它的作用完全取决于人们掌握和利用这些信息的方式。有的人掌握的信息多（通常是专家），有的人掌握的信息少（普通大众），用经济学术语说，这种现象被称为信息不对称。信息不对称是有优势的，比如不良医生可以利用信息不对称让病人多花冤枉钱，或者二手车卖主隐瞒车子的实际情况卖出更高价格等等。虽说借助互联网的传播特性，信息不对称的现象在减少，但某种程度上说也可以说并没有减少，因为信息传播变容易了，那么假信息传播也变容易了（这是另外一个话题，不展开）。那些所谓的专家必须得保持信息不对称才能获取某种利益（金钱、尊重或者权力），获取利益的方式就是使用诉诸恐惧。

我们都听过这样一句话“人会害怕，往往是因为未知”，这一句话就把上面提到的“信息不对称”和“诉诸恐惧”给包含进去了。我们以学习一门PPT技能为例，刚开始的时候肯定会觉得非常难，好多东西，一头雾水，不知道该学什么（信息不对称），然后因为畏难情绪（诉诸恐惧）就不学了，自己把自己吓倒了。我们要做的就是正确认识到害怕现象是正常的，我们采取的措施就是立马开始做，要不找到一个PPT牛人让他简要介绍关于PPT的知识，要不自己上网查阅，消除信息不对称，这样自己有了了解就不会害怕了。

结论

这两本书对我最大的启发就是经济学思维，把所有的事物标上“价格”，适当地加入动机思考，使用科学研究的方法对数据进行“提纯”，提取出直观的信息。最关键的一点是：使用大数据分析加上逻辑思考找出事物背后的关联。这两本书也有其缺陷之处，每一章的主题不是很明确，作者经常举各种例子，例子还特别详细，看得时候很容易忘记这一章到底在讲什么，这个时候需要读者自己把例子精简，纵观全章得出结论。

后记

写这篇文章是一个痛苦的过程，写作实际上是一种真实面对自己内心的活动，你会看到自己的逻辑，你会思考该怎么讲述（信息那一节我一直没想到该如何安排），你会想到很多东西但是又不得不放弃（与主题关联不是很大）。在说关联的时候，有一个隐藏的关键词“逻辑”，逻辑太重要了，但是我讲不好它只好放弃。科学研究方法可以继续展开，逻辑也可以展开。这篇文章有点虎头蛇尾的感觉，不知道该如何收尾。不过这也是成长的一部分，坦然接受吧，然后多多观察别人如何组织文章不断进步就行了。
